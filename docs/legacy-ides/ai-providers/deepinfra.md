# Using DeepInfra With Kilo Code

The final legacy IDE extensions include a DeepInfra provider configuration.

## Configuration

1. Open Kilo Code settings using the gear icon in the Kilo Code panel.
2. Select **DeepInfra** from the **API Provider** dropdown.
3. Configure a DeepInfra API key and model.
4. Select a model when the provider exposes a model picker, then save the settings.

Provider account and API-key management: [https://deepinfra.com/](https://deepinfra.com/)

API keys are stored using the extension's secure secret storage.
