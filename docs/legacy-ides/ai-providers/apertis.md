# Using Apertis With Kilo Code

The final legacy IDE extensions include a Apertis provider configuration.

## Configuration

1. Open Kilo Code settings using the gear icon in the Kilo Code panel.
2. Select **Apertis** from the **API Provider** dropdown.
3. Configure an Apertis API key.
4. Select a model when the provider exposes a model picker, then save the settings.

Provider account and API-key management: [https://apertis.ai/token](https://apertis.ai/token)

API keys are stored using the extension's secure secret storage.
