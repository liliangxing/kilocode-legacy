# Using SambaNova With Kilo Code

The final legacy IDE extensions include a SambaNova provider configuration.

## Configuration

1. Open Kilo Code settings using the gear icon in the Kilo Code panel.
2. Select **SambaNova** from the **API Provider** dropdown.
3. Configure a SambaNova Cloud API key.
4. Select a model when the provider exposes a model picker, then save the settings.

Provider account and API-key management: [https://cloud.sambanova.ai/](https://cloud.sambanova.ai/)

API keys are stored using the extension's secure secret storage.
