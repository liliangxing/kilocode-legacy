# Using IO Intelligence With Kilo Code

The final legacy IDE extensions include a IO Intelligence provider configuration.

## Configuration

1. Open Kilo Code settings using the gear icon in the Kilo Code panel.
2. Select **IO Intelligence** from the **API Provider** dropdown.
3. Configure an IO Intelligence API key and model.
4. Select a model when the provider exposes a model picker, then save the settings.

Provider account and API-key management: [https://ai.io.net/ai/api-keys](https://ai.io.net/ai/api-keys)

API keys are stored using the extension's secure secret storage.
