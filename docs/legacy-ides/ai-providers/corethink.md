# Using CoreThink With Kilo Code

The final legacy IDE extensions include a CoreThink provider configuration.

## Configuration

1. Open Kilo Code settings using the gear icon in the Kilo Code panel.
2. Select **CoreThink** from the **API Provider** dropdown.
3. Configure the provider options shown in Kilo Code.
4. Select a model when the provider exposes a model picker, then save the settings.

Provider account and API-key management: [https://corethink.ai/](https://corethink.ai/)

API keys are stored using the extension's secure secret storage.
