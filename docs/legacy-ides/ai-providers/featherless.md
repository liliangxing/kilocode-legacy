# Using Featherless AI With Kilo Code

The final legacy IDE extensions include a Featherless AI provider configuration.

## Configuration

1. Open Kilo Code settings using the gear icon in the Kilo Code panel.
2. Select **Featherless AI** from the **API Provider** dropdown.
3. Configure a Featherless API key.
4. Select a model when the provider exposes a model picker, then save the settings.

Provider account and API-key management: [https://featherless.ai/account/api-keys](https://featherless.ai/account/api-keys)

API keys are stored using the extension's secure secret storage.
