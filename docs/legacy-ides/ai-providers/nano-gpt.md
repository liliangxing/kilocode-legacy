# Using Nano-GPT With Kilo Code

The final legacy IDE extensions include a Nano-GPT provider configuration.

## Configuration

1. Open Kilo Code settings using the gear icon in the Kilo Code panel.
2. Select **Nano-GPT** from the **API Provider** dropdown.
3. Configure a Nano-GPT API key, model-list filter, and model.
4. Select a model when the provider exposes a model picker, then save the settings.

Provider account and API-key management: [https://nano-gpt.com/](https://nano-gpt.com/)

API keys are stored using the extension's secure secret storage.
