# Using Doubao With Kilo Code

The final legacy IDE extensions include a Doubao provider configuration.

## Configuration

1. Open Kilo Code settings using the gear icon in the Kilo Code panel.
2. Select **Doubao** from the **API Provider** dropdown.
3. Configure a Volcengine Ark API key.
4. Select a model when the provider exposes a model picker, then save the settings.

Provider account and API-key management: [https://www.volcengine.com/experience/ark](https://www.volcengine.com/experience/ark)

API keys are stored using the extension's secure secret storage.
