# Using Hugging Face With Kilo Code

The final legacy IDE extensions include a Hugging Face provider configuration.

## Configuration

1. Open Kilo Code settings using the gear icon in the Kilo Code panel.
2. Select **Hugging Face** from the **API Provider** dropdown.
3. Configure a Hugging Face token, model, and inference provider.
4. Select a model when the provider exposes a model picker, then save the settings.

Provider account and API-key management: [https://huggingface.co/settings/tokens](https://huggingface.co/settings/tokens)

API keys are stored using the extension's secure secret storage.
